# Agregar y seleccionar

Seleccione múltiples elementos en listas o el diagrama de Gantt, y controle dónde se insertan los nuevos elementos.

## Seleccionar múltiples elementos

En **Android** e **iOS**, use una pulsación larga para seleccionar elementos adicionales. Mantenga el dedo sobre un elemento más tiempo que un toque normal.

En **Windows**, **macOS** y **Web**, use los siguientes métodos:

- Mantenga `Ctrl` mientras hace clic para seleccionar múltiples elementos individuales.
- Mantenga `Shift` mientras hace clic para seleccionar un rango de elementos desde la selección actual hasta el elemento clicado.

La selección múltiple funciona en todas las listas (tareas, recursos, calendarios) y en las barras de tareas en el diagrama de Gantt.

## Agregar e insertar elementos

El botón **+** agrega o inserta elementos dependiendo de su selección actual.

- Si un elemento (tarea, recurso o calendario) está seleccionado, el nuevo elemento se inserta debajo de él.
- Si múltiples elementos están seleccionados, el nuevo elemento se inserta debajo del último elemento seleccionado.
- Si no hay elementos seleccionados, el nuevo elemento se agrega al final de la lista.
