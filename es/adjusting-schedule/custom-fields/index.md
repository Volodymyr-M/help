# Campos personalizados

Agregue sus propias columnas de datos a las tareas — rastree departamento, códigos de prioridad, estado de aprobación o cualquier información específica del proyecto junto con los campos integrados.

Ingantt admite campos personalizados en tareas, incluyendo tipos de campo Texto, Número, Indicador, Fecha, Inicio, Fin, Duración y Costo.

Para gestionar campos personalizados, abra el elemento **Campos personalizados** en el menú **Proyecto**. Puede agregar definiciones de campos personalizados, establecer alias (nombres para mostrar) y configurar tablas de búsqueda con valores predefinidos.

Para editar valores de campos personalizados, use la pestaña **Campos personalizados** del diálogo **Propiedades de la tarea**. Para agregar columnas de campos personalizados a la tabla de tareas, use el diálogo **Opciones**.

Los campos personalizados también admiten fórmulas (evaluadas automáticamente) y acumulación a [tareas de resumen](/es/building-schedule/tasks/index.md#tareas-de-resumen) — Suma, Máximo, Mínimo, Conteo y Promedio.

## Códigos de esquema

Los códigos de esquema son campos de clasificación jerárquica definidos por el usuario para tareas (por ejemplo, códigos de departamento, códigos de ubicación). Adminístrelos desde el elemento **Códigos de esquema** en el menú **Proyecto**.

Cada código de esquema tiene una máscara de código que define el formato por nivel de jerarquía y una tabla de búsqueda de valores permitidos. Asigne valores de códigos de esquema en la pestaña **Códigos de esquema** del diálogo **Propiedades de la tarea**. Puede agregar las columnas correspondientes a la tabla de tareas.
