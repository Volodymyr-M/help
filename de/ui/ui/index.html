Hinzufügen und Auswählen #

Wählen Sie mehrere Elemente in Listen oder im Gantt-Diagramm aus und steuern Sie, wo neue Elemente eingefügt werden.

Mehrere Elemente auswählen #

Auf Android und iOS verwenden Sie einen langen Druck, um zusätzliche Elemente auszuwählen. Halten Sie Ihren Finger länger als bei einem normalen Tippen auf einem Element.

Auf Windows, macOS und im Web verwenden Sie die folgenden Methoden:

  • Halten Sie Strg gedrückt, während Sie klicken, um mehrere einzelne Elemente auszuwählen.
  • Halten Sie Umschalt gedrückt, während Sie klicken, um einen Bereich von Elementen von der aktuellen Auswahl bis zum angeklickten Element auszuwählen.

Die Mehrfachauswahl funktioniert in allen Listen (Vorgänge, Ressourcen, Kalender) und auf Vorgangsbalken im Gantt-Diagramm.

Elemente hinzufügen und einfügen #

Die +-Schaltfläche fügt Elemente hinzu oder fügt sie ein, abhängig von Ihrer aktuellen Auswahl.

  • Wenn ein Element (Vorgang, Ressource oder Kalender) ausgewählt ist, wird das neue Element darunter eingefügt.
  • Wenn mehrere Elemente ausgewählt sind, wird das neue Element unterhalb des zuletzt ausgewählten Elements eingefügt.
  • Wenn keine Elemente ausgewählt sind, wird das neue Element am Ende der Liste hinzugefügt.
