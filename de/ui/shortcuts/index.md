# Tastaturkürzel

Beschleunigen Sie Ihre Arbeit mit Tastaturkürzeln für häufige Aktionen wie Bearbeiten, Navigieren und Ansichten wechseln.

> Tastaturkürzel sind auf **Android** und **iOS** nicht verfügbar.
>
> Auf **macOS** verwenden Sie `Cmd` anstelle von `Strg`.

| Tastenkürzel               | Beschreibung                                                                                                                                       |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| `Strg + X`             | Ausgewählte Elemente in die Zwischenablage ausschneiden.                                                                                                              |
| `Strg + C`             | Ausgewählte Elemente in die Zwischenablage kopieren.                                                                                                             |
| `Strg + V`             | Elemente aus der Zwischenablage unterhalb des ausgewählten Elements einfügen, oder am Ende der Liste, wenn nichts ausgewählt ist.                                         |
| `Strg + Z`             | Letzte Aktion rückgängig machen.                                                                                                                             |
| `Strg + Y`             | Letzte rückgängig gemachte Aktion wiederherstellen.                                                                                                                      |
| `Strg + S`             | Änderungen in einer Datei speichern.                                                                                                                           |
| `Strg + O`             | Datei öffnen (native Dateiauswahl auf dem Desktop, Dialog **Datei öffnen** im **Web**).                                                                     |
| `Strg + P`             | Aktuelles Projekt drucken.                                                                                                                        |
| `Strg + Umschalt + F`     | Sichtbarkeit der **Kopfzeile/Menüleiste** umschalten.                                                                                                            |
| `A`                    | Neues Element hinzufügen.                                                                                                                   |
| `Entf`                  | Ausgewählte Elemente löschen.                                                                                                                            |
| `Eingabe`                | Dialog **Eigenschaften** für das ausgewählte Element anzeigen. Bei Mehrfachauswahl wird der Dialog für das oberste Element angezeigt.                       |
| `Pfeil hoch`             | Vorheriges Element auswählen.                                                                                                                         |
| `Pfeil runter`           | Nächstes Element auswählen.                                                                                                                             |
| `Umschalt + Pfeil hoch/runter` | Auswahl in einer Listenansicht nach oben oder unten erweitern.                                                                                                |
| `Strg + Pfeil hoch`      | Ausgewählte Vorgänge nach oben verschieben.                                                                                                                           |
| `Strg + Pfeil runter`    | Ausgewählte Vorgänge nach unten verschieben.                                                                                                                         |
| `Strg + Pfeil links`    | Einzug der ausgewählten Vorgänge verkleinern.                                                                                                                |
| `Strg + Pfeil rechts`   | Einzug der ausgewählten Vorgänge vergrößern.                                                                                                                |
| `Tab / Umschalt + Tab`    | Zur nächsten oder vorherigen bearbeitbaren Zelle wechseln. Bestätigt die aktuelle Bearbeitung vor dem Wechsel.                                                               |
| `R`                    | Zur Ansicht **Ressourcen** wechseln.                                                                                                                     |
| `C`                    | Zur Ansicht **Kalender** wechseln.                                                                                                                     |
| `T`                    | Zur Ansicht **Vorgänge** wechseln.                                                                                                                         |
| `P`                    | Dialog **Projekteigenschaften** anzeigen.                                                                                                           |
| `I`                    | Gantt-Diagramm vergrößern.                                                                                                                       |
| `O`                    | Gantt-Diagramm verkleinern.                                                                                                                      |
| `L`                    | Ausgewählte Vorgänge verknüpfen.                                                                                                                              |
| `U`                    | Verknüpfung ausgewählter Vorgänge aufheben (Abhängigkeit zwischen ihnen entfernen).                                                                                       |
| `Ziffernblock +`             | Gantt-Diagramm vergrößern.                                                                                                                       |
| `Ziffernblock -`             | Gantt-Diagramm verkleinern.                                                                                                                      |
| `Esc`                  | Vollbildmodus beenden; Dialoge abbrechen.                                                                                                            |
| `Strg + Eingabe`         | Aktuellen Dialog bestätigen.                                                                                                                       |
| `Strg + Alt + Umschalt + H` | Versionshistorie anzeigen.                                                                                                                           |
