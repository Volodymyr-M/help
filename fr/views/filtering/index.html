Filtrage des tâches #

Concentrez-vous sur l'essentiel en masquant les tâches que vous n'avez pas besoin de voir pour le moment — travail terminé, tâches non critiques ou tâches non affectées à certaines personnes.

L'option Filter dans le menu View vous permet de filtrer les tâches affichées dans la vue des tâches et le diagramme de Gantt.

Les options de filtre incluent :

  • Hide complete tasks — Masquer les tâches achevées à 100 %
  • Hide non-critical tasks — Afficher uniquement les tâches du chemin critique (et leurs tâches récapitulatives parentes)
  • Filter by resource — Afficher uniquement les tâches affectées à des ressources spécifiques

Pour appliquer un filtre :

  1. Choisissez Filter dans le menu View
  2. Cochez les options de filtre souhaitées
  3. Sélectionnez une ou plusieurs ressources pour filtrer (facultatif)
  4. Cliquez sur OK pour appliquer le filtre

Lorsqu'un filtre est actif, seules les tâches correspondantes sont affichées dans la liste des tâches et le diagramme de Gantt. Le bouton de filtre dans la barre d'outils est mis en évidence lorsqu'un filtre est actif. Utilisez Clear All dans la boîte de dialogue Filter pour réinitialiser tous les filtres.

Les filtres s'appliquent à toutes les vues basées sur les tâches : Tasks, Tracking Gantt, Task Board, Network Diagram, Calendar View, et plus encore.

Le filtrage n'affecte que l'affichage ; il ne supprime ni ne modifie vos tâches.
