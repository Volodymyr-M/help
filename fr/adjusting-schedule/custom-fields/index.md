# Champs personnalisés

Ajoutez vos propres colonnes de données aux tâches — suivez le département, les codes de priorité, le statut d'approbation ou toute information spécifique au projet en complément des champs intégrés.

Ingantt prend en charge les champs personnalisés sur les tâches, avec les types Text, Number, Flag, Date, Start, Finish, Duration et Cost.

Pour gérer les champs personnalisés, ouvrez l'élément **Custom Fields** dans le menu **Project**. Vous pouvez ajouter des définitions de champs personnalisés, définir des alias (noms d'affichage) et configurer des tables de correspondance avec des valeurs prédéfinies.

Pour modifier les valeurs des champs personnalisés, utilisez l'onglet **Custom Fields** de la boîte de dialogue **Task Properties**. Pour ajouter des colonnes de champs personnalisés au tableau des tâches, utilisez la boîte de dialogue **Options**.

Les champs personnalisés prennent également en charge les formules (évaluées automatiquement) et la consolidation vers les [tâches récapitulatives](/fr/building-schedule/tasks/index.md#summary-tasks) — Sum, Max, Min, Count et Average.

## Codes hiérarchiques

Les codes hiérarchiques sont des champs de classification hiérarchique définis par l'utilisateur pour les tâches (par exemple, codes de département, codes de localisation). Gérez-les depuis l'élément **Outline Codes** dans le menu **Project**.

Chaque code hiérarchique possède un masque de code qui définit le format par niveau hiérarchique et une table de correspondance des valeurs autorisées. Attribuez les valeurs des codes hiérarchiques dans l'onglet **Outline Codes** de la boîte de dialogue **Task Properties**. Vous pouvez ajouter les colonnes correspondantes au tableau des tâches.
