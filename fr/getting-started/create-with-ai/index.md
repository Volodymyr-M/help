# Créer avec l'IA

Laissez l'IA construire un plan de projet complet pour vous — décrivez simplement votre projet et Ingantt génère les tâches, les dépendances et un calendrier en quelques secondes.

## Créer un projet

Pour créer un planning de projet avec l'IA :

1. Cliquez sur le bouton **+** sur l'écran des projets. Si vous êtes déjà en train de modifier un projet, ouvrez plutôt le menu **File**.
2. Choisissez **Create new with AI**.
3. Saisissez une description de votre projet dans la boîte de dialogue.
4. Cliquez sur **CREATE**.

![Create new with AI](/images/getting-started/create_with_ai.png)

## Rédiger une bonne description

Votre description peut être aussi générique que _« Développement d'une application iOS de livraison de repas »_ ou aussi détaillée que vous le souhaitez.

Par exemple, le projet de rénovation que [Premiers pas](/fr/getting-started/getting-started/index.md) présente étape par étape produit le même planning à partir d'une description telle que :

> _« Mon projet de rénovation » commence le 1er janvier 2027. Le peintre peint les murs pendant 2 jours, puis le menuisier installe les meubles de cuisine pendant 2 jours. Le peintre a un jour de congé le 4 janvier 2027. Considérez le 6 janvier 2027 comme un jour férié pour tout le monde._

Si votre projet est volumineux, essayez de le découper en sous-projets et de générer un plan distinct pour chacun.

Les résultats peuvent varier d'une exécution à l'autre ; si le résultat ne vous convient pas, réessayez.
