Créer avec l'IA #

Laissez l'IA construire un plan de projet complet pour vous — décrivez simplement votre projet et Ingantt génère les tâches, les dépendances et un calendrier en quelques secondes. Pour créer un planning de projet avec l'IA :

  1. Cliquez sur le bouton + sur l'écran des projets.
  2. Choisissez Create new with AI.
  3. Saisissez une description de votre projet dans la boîte de dialogue.
  4. Cliquez sur CREATE.

Votre description peut être aussi générique que « Développement d'une application iOS de livraison de repas » ou aussi détaillée que vous le souhaitez. Si votre projet est volumineux, essayez de le découper en sous-projets et de générer un plan distinct pour chacun. Les résultats peuvent varier d'une exécution à l'autre ; si le résultat ne vous convient pas, réessayez.
