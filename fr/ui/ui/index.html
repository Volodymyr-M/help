Ajout et sélection #

Sélectionnez plusieurs éléments dans les listes ou le diagramme de Gantt, et choisissez où insérer les nouveaux éléments.

Sélectionner plusieurs éléments #

Sur Android et iOS, utilisez un appui long pour sélectionner des éléments supplémentaires. Maintenez votre doigt sur un élément plus longtemps qu'un appui normal.

Sur Windows, macOS et Web, utilisez les méthodes suivantes :

  • Maintenez Ctrl enfoncé tout en cliquant pour sélectionner plusieurs éléments individuels.
  • Maintenez Shift enfoncé tout en cliquant pour sélectionner une plage d'éléments depuis la sélection actuelle jusqu'à l'élément cliqué.

La sélection multiple fonctionne dans toutes les listes (tâches, ressources, calendriers) et sur les barres de tâches du diagramme de Gantt.

Ajouter et insérer des éléments #

Le bouton + ajoute ou insère des éléments en fonction de votre sélection actuelle.

  • Si un élément (tâche, ressource ou calendrier) est sélectionné, le nouvel élément est inséré en dessous.
  • Si plusieurs éléments sont sélectionnés, le nouvel élément est inséré sous le dernier élément sélectionné.
  • Si aucun élément n'est sélectionné, le nouvel élément est ajouté à la fin de la liste.
