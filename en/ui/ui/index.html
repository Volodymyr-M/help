Adding and Selecting #

Select multiple items in lists or the Gantt chart, and control where new items are inserted.

Select Multiple Items #

On Android and iOS, use a long press to select additional items. Hold your finger on an item longer than a normal tap.

On Windows, macOS, and Web, use the following methods:

  • Hold Ctrl while clicking to select multiple individual items.
  • Hold Shift while clicking to select a range of items from the current selection to the clicked item.

Multi-selection works in all lists (tasks, resources, calendars) and on task bars in the Gantt chart.

Add and Insert Items #

The + button adds or inserts items depending on your current selection.

  • If an item (task, resource, or calendar) is selected, the new item is inserted below it.
  • If multiple items are selected, the new item is inserted below the last selected item.
  • If no items are selected, the new item is added to the end of the list.
