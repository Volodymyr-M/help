Keyboard Shortcuts #

Speed up your work with keyboard shortcuts for common actions like editing, navigating, and switching views.

Keyboard shortcuts are not available on Android and iOS.

On macOS, use Cmd instead of Ctrl.

Shortcut Description
Ctrl + X Cut selected items to the clipboard.
Ctrl + C Copy selected items to the clipboard.
Ctrl + V Paste items from the clipboard below the selected item, or at the end of the list if nothing is selected.
Ctrl + Z Undo the last action.
Ctrl + Y Redo the last undone action.
Ctrl + S Save changes to a file.
Ctrl + O Open a file (native file picker on desktop, Open file dialog on Web).
Ctrl + P Print the current project.
Ctrl + Shift + F Toggle header/menu bar visibility.
A Add a new item.
Del Delete selected items.
Enter Show the Properties dialog for the selected item. If multiple items are selected, the dialog is shown for the top item.
Arrow Up Select the previous item.
Arrow Down Select the next item.
Shift + Arrow Up/Down Extend the selection up or down in any list view.
Ctrl + Arrow Up Move selected tasks up.
Ctrl + Arrow Down Move selected tasks down.
Ctrl + Arrow Left Decrease indent of selected tasks.
Ctrl + Arrow Right Increase indent of selected tasks.
Tab / Shift + Tab Move to the next or previous editable cell. Commits the current edit before moving.
R Switch to Resources view.
C Switch to Calendars view.
T Switch to Tasks view.
P Show the Project Properties dialog.
I Zoom in on the Gantt chart.
O Zoom out on the Gantt chart.
L Link selected tasks.
U Unlink selected tasks (remove the dependency between them).
Numpad + Zoom in on the Gantt chart.
Numpad - Zoom out on the Gantt chart.
Esc Exit full-screen mode; cancel dialogs.
Ctrl + Enter Confirm the current dialog.
Ctrl + Alt + Shift + H Show version history.
