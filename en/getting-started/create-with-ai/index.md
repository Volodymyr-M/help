# Create with AI

Let AI build a full project plan for you — just describe your project and Ingantt generates tasks, dependencies, and a timeline in seconds.
To create a project schedule with AI:

1. Click the **+** button on the Projects screen.
2. Choose **Create new with AI**.
3. Enter a description of your project in the dialog box.
4. Click **CREATE**.

> Your description can be as generic as _"Development of an iOS food delivery app"_ or as detailed as you like. If your project is large, try splitting it into sub-projects and generating separate project plans for each one. Results may vary between runs, so if you're not happy with the output, try again.
