Filtering Tasks #

Focus on what matters by hiding tasks you don't need to see right now — completed work, non-critical tasks, or tasks not assigned to specific people.

The Filter option in the View menu allows you to filter which tasks are displayed in the task view and Gantt chart.

Filter options include:

  • Hide complete tasks — Hide tasks that are 100% complete
  • Hide non-critical tasks — Show only tasks on the critical path (and their ancestor summary tasks)
  • Filter by resource — Show only tasks assigned to specific resources

To apply a filter:

  1. Choose Filter from the View menu
  2. Check the desired filter options
  3. Select one or more resources to filter by (optional)
  4. Click OK to apply the filter

When a filter is active, only matching tasks are shown in the task list and Gantt chart. The filter button in the toolbar is highlighted when any filter is active. Use Clear All in the Filter dialog to reset all filters.

Filters apply across all task-based views: Tasks, Tracking Gantt, Task Board, Network Diagram, Calendar View, and more.

Filtering affects only the display; it does not delete or modify your tasks.
