# Custom Fields

Add your own data columns to tasks — track department, priority codes, approval status, or any project-specific information alongside the built-in fields.

Ingantt supports custom fields on tasks, including Text, Number, Flag, Date, Start, Finish, Duration, and Cost field types.

To manage custom fields, open the **Custom Fields** item in the **Project** menu. You can add custom field definitions, set aliases (display names), and configure lookup tables with predefined values.

To edit custom field values, use the **Custom Fields** tab of the **Task Properties** dialog. To add custom field columns to the task table, use the **Options** dialog.

Custom fields also support formulas (evaluated automatically) and rollup to [summary tasks](/en/building-schedule/tasks/index.md#summary-tasks) — Sum, Max, Min, Count, and Average.

## Outline Codes

Outline codes are user-defined hierarchical classification fields for tasks (e.g., department codes, location codes). Manage them from the **Outline Codes** item in the **Project** menu.

Each outline code has a code mask that defines the format per hierarchy level and a lookup table of allowed values. Assign outline code values on the **Outline Codes** tab of the **Task Properties** dialog. You can add corresponding columns to the task table.
