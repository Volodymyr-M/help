# Фільтрація завдань

Зосередьтеся на важливому, приховавши завдання, які вам зараз не потрібні — завершену роботу, некритичні завдання або завдання, не призначені певним людям.

Пункт **Фільтр** у меню **Подання** дозволяє фільтрувати, які завдання відображаються в поданні завдань та на діаграмі Ганта.

Параметри фільтрації включають:

- **Приховати завершені завдання** — Приховати завдання, що виконані на 100%
- **Приховати некритичні завдання** — Показати лише завдання на критичному шляху (та їхні батьківські сумарні завдання)
- **Фільтр за ресурсом** — Показати лише завдання, призначені конкретним ресурсам

Щоб застосувати фільтр:

1. Оберіть **Фільтр** з меню **Подання**
2. Позначте потрібні параметри фільтрації
3. Виберіть один або кілька ресурсів для фільтрації (необов'язково)
4. Натисніть **ОК** для застосування фільтра

Коли фільтр активний, у списку завдань та на діаграмі Ганта відображаються лише відповідні завдання. Кнопка фільтра на панелі інструментів виділяється, коли будь-який фільтр активний. Використовуйте **Очистити все** в діалозі фільтра для скидання всіх фільтрів.

Фільтри застосовуються до всіх подань на основі завдань: Завдання, Діаграма Ганта для відстеження, Дошка завдань, Мережева діаграма, Календарне подання та інші.

> Фільтрація впливає лише на відображення; вона не видаляє та не змінює ваші завдання.
