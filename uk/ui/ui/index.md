# Додавання та вибір

Вибирайте кілька елементів у списках або на діаграмі Ганта та контролюйте, де вставляються нові елементи.

## Вибір кількох елементів

На **Android** та **iOS** використовуйте довге натискання для вибору додаткових елементів. Утримуйте палець на елементі довше, ніж при звичайному натисканні.

На **Windows**, **macOS** та у **веб-версії** використовуйте наступні способи:

- Утримуйте `Ctrl` під час натискання для вибору кількох окремих елементів.
- Утримуйте `Shift` під час натискання для вибору діапазону елементів від поточного вибору до натиснутого елемента.

Множинний вибір працює в усіх списках (завдання, ресурси, календарі) та на смугах завдань на діаграмі Ганта.

## Додавання та вставка елементів

Кнопка **+** додає або вставляє елементи залежно від вашого поточного вибору.

- Якщо елемент (завдання, ресурс або календар) вибраний, новий елемент вставляється під ним.
- Якщо вибрано кілька елементів, новий елемент вставляється під останнім вибраним.
- Якщо жоден елемент не вибраний, новий елемент додається в кінець списку.
