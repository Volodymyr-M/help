# Клавіатурні скорочення

Прискоріть свою роботу за допомогою клавіатурних скорочень для поширених дій, таких як редагування, навігація та перемикання подань.

> Клавіатурні скорочення недоступні на **Android** та **iOS**.
>
> На **macOS** використовуйте `Cmd` замість `Ctrl`.

| Скорочення | Опис |
|------------|------|
| `Ctrl + X` | Вирізати вибрані елементи до буфера обміну. |
| `Ctrl + C` | Копіювати вибрані елементи до буфера обміну. |
| `Ctrl + V` | Вставити елементи з буфера обміну під вибраним елементом або в кінець списку, якщо нічого не вибрано. |
| `Ctrl + Z` | Скасувати останню дію. |
| `Ctrl + Y` | Повторити останню скасовану дію. |
| `Ctrl + S` | Зберегти зміни у файл. |
| `Ctrl + O` | Відкрити файл (стандартний вибір файлу на десктопі, діалог **Відкрити файл** у **веб-версії**). |
| `Ctrl + P` | Друк поточного проекту. |
| `Ctrl + Shift + F` | Перемкнути видимість **заголовка/рядка меню**. |
| `A` | Додати новий елемент. |
| `Del` | Видалити вибрані елементи. |
| `Enter` | Показати діалог **Властивості** для вибраного елемента. Якщо вибрано кілька елементів, діалог показується для верхнього. |
| `Стрілка вгору` | Вибрати попередній елемент. |
| `Стрілка вниз` | Вибрати наступний елемент. |
| `Shift + Стрілка вгору/вниз` | Розширити вибір вгору або вниз у будь-якому поданні списку. |
| `Ctrl + Стрілка вгору` | Перемістити вибрані завдання вгору. |
| `Ctrl + Стрілка вниз` | Перемістити вибрані завдання вниз. |
| `Ctrl + Стрілка вліво` | Зменшити відступ вибраних завдань. |
| `Ctrl + Стрілка вправо` | Збільшити відступ вибраних завдань. |
| `Tab / Shift + Tab` | Перейти до наступної або попередньої комірки для редагування. Зберігає поточне редагування перед переходом. |
| `R` | Переключитись на подання **Ресурси**. |
| `C` | Переключитись на подання **Календарі**. |
| `T` | Переключитись на подання **Завдання**. |
| `P` | Показати діалог **Властивості проекту**. |
| `I` | Збільшити масштаб діаграми Ганта. |
| `O` | Зменшити масштаб діаграми Ганта. |
| `L` | Зв'язати вибрані завдання. |
| `U` | Роз'єднати вибрані завдання (видалити залежність між ними). |
| `Numpad +` | Збільшити масштаб діаграми Ганта. |
| `Numpad -` | Зменшити масштаб діаграми Ганта. |
| `Esc` | Вийти з повноекранного режиму; скасувати діалоги. |
| `Ctrl + Enter` | Підтвердити поточний діалог. |
| `Ctrl + Alt + Shift + H` | Показати історію версій. |
