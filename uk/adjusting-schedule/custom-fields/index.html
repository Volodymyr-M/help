Настроювані поля #

Додайте власні стовпці даних до завдань — відстежуйте відділ, коди пріоритету, статус затвердження або будь-яку специфічну для проекту інформацію разом із вбудованими полями.

Ingantt підтримує настроювані поля для завдань, включаючи типи полів: Текст, Число, Прапорець, Дата, Початок, Завершення, Тривалість та Вартість.

Щоб керувати настроюваними полями, відкрийте пункт Настроювані поля у меню Проект. Ви можете додавати визначення настроюваних полів, встановлювати псевдоніми (назви для відображення) та налаштовувати таблиці підстановки з попередньо визначеними значеннями.

Щоб редагувати значення настроюваних полів, використовуйте вкладку Настроювані поля діалогу Властивості завдання. Щоб додати стовпці настроюваних полів до таблиці завдань, використовуйте діалог Параметри.

Настроювані поля також підтримують формули (обчислюються автоматично) та підсумки для сумарних завдань — Сума, Максимум, Мінімум, Кількість та Середнє.

Коди структури #

Коди структури — це визначені користувачем ієрархічні класифікаційні поля для завдань (наприклад, коди відділів, коди розташування). Керуйте ними з пункту Коди структури у меню Проект.

Кожен код структури має маску коду, що визначає формат для кожного рівня ієрархії, та таблицю підстановки з дозволеними значеннями. Призначайте значення кодів структури на вкладці Коди структури діалогу Властивості завдання. Ви можете додати відповідні стовпці до таблиці завдань.
