Освоєний обсяг #

Управління освоєним обсягом (УОО) поєднує вимірювання обсягу, розкладу та вартості для оцінки того, чи проект вкладається в графік та бюджет. Встановіть дату стану та базовий план, і Ingantt розрахує стандартні галузеві метрики УОО.

Дата стану #

Дата стану визначає дату, що використовується для відстеження прогресу та розрахунків освоєного обсягу. Коли встановлена, на діаграмі Ганта відображається зелена вертикальна лінія на цій даті.

Коли дата стану не встановлена, для розрахунків використовується поточна дата. Ви можете встановити дату стану в діалозі Параметри освоєного обсягу, доступному з меню Проект.

Управління освоєним обсягом #

Ingantt підтримує управління освоєним обсягом — методику управління проектами, що інтегрує дані обсягу, розкладу та вартості для оцінки ефективності проекту.

Налаштування освоєного обсягу #

  1. Створіть базовий план для фіксації запланованого розкладу та витрат.
  2. Встановіть Дату стану в діалозі Параметри освоєного обсягу (доступному з меню Проект) або залиште стандартне значення — поточну дату.
  3. Оновлюйте значення % виконання завдань у міру виконання робіт.

Доступні метрики УОО #

Наступні стовпці УОО можна додати до списку завдань через діалог Параметри:

Метрика Опис
БВЗР Бюджетна вартість запланованої роботи (Планова вартість)
БВВР Бюджетна вартість виконаної роботи (Освоєний обсяг)
ФВВР Фактична вартість виконаної роботи
БПЗ Бюджет по завершенні
ОПЗ Оцінка по завершенні
ВПЗ Відхилення по завершенні
ВР Відхилення розкладу
ВВ Відхилення вартості
ІВР Індекс виконання розкладу
ІВВ Індекс виконання вартості
ІВДЗ Індекс виконання до завершення
ВР% Відхилення розкладу у відсотках
ВВ% Відхилення вартості у відсотках

Метод освоєного обсягу #

Кожне завдання може використовувати один із двох методів для розрахунку БВВР (Освоєного обсягу):

  • % виконання (за замовчуванням) — Використовує відсоток виконання завдання для розрахунку освоєного обсягу.
  • Фізичний % виконання — Використовує окремо відстежуваний фізичний відсоток виконання. Корисний для вимірювання прогресу на основі результатів.

Встановіть метод для окремих завдань на вкладці Вартість та ОО діалогу Властивості завдання. Змініть стандартний метод для нових завдань у діалозі Параметри освоєного обсягу.

Базовий план для освоєного обсягу #

Розрахунки УОО використовують конкретний базовий план. Ви можете обрати, який базовий план (від Базовий план 0 до Базовий план 10) використовується для освоєного обсягу в діалозі Параметри освоєного обсягу. За замовчуванням використовується Базовий план 0.
