# Scorciatoie da tastiera

Velocizza il tuo lavoro con le scorciatoie da tastiera per le azioni comuni come modifica, navigazione e cambio di vista.

> Le scorciatoie da tastiera non sono disponibili su **Android** e **iOS**.
>
> Su **macOS**, usa `Cmd` al posto di `Ctrl`.

| Scorciatoia               | Descrizione                                                                                                                                       |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| `Ctrl + X`             | Taglia gli elementi selezionati negli appunti.                                                                                                              |
| `Ctrl + C`             | Copia gli elementi selezionati negli appunti.                                                                                                             |
| `Ctrl + V`             | Incolla gli elementi dagli appunti sotto l'elemento selezionato, o alla fine dell'elenco se nulla è selezionato.                                         |
| `Ctrl + Z`             | Annulla l'ultima azione.                                                                                                                             |
| `Ctrl + Y`             | Ripeti l'ultima azione annullata.                                                                                                                      |
| `Ctrl + S`             | Salva le modifiche in un file.                                                                                                                           |
| `Ctrl + O`             | Apri un file (selezione file nativa su desktop, finestra **Open file** su **Web**).                                                                     |
| `Ctrl + P`             | Stampa il progetto corrente.                                                                                                                        |
| `Ctrl + Shift + F`     | Mostra/nascondi la visibilità dell'**intestazione/barra dei menu**.                                                                                                            |
| `A`                    | Aggiungi un nuovo elemento.                                                                                                                                   |
| `Del`                  | Elimina gli elementi selezionati.                                                                                                                            |
| `Enter`                | Mostra la finestra **Properties** per l'elemento selezionato. Se sono selezionati più elementi, la finestra viene mostrata per il primo elemento.                       |
| `Arrow Up`             | Seleziona l'elemento precedente.                                                                                                                         |
| `Arrow Down`           | Seleziona l'elemento successivo.                                                                                                                             |
| `Shift + Arrow Up/Down` | Estendi la selezione verso l'alto o il basso in qualsiasi vista a elenco.                                                                                                |
| `Ctrl + Arrow Up`      | Sposta le attività selezionate verso l'alto.                                                                                                                           |
| `Ctrl + Arrow Down`    | Sposta le attività selezionate verso il basso.                                                                                                                         |
| `Ctrl + Arrow Left`    | Riduci il rientro delle attività selezionate.                                                                                                                |
| `Ctrl + Arrow Right`   | Aumenta il rientro delle attività selezionate.                                                                                                                |
| `Tab / Shift + Tab`    | Passa alla cella modificabile successiva o precedente. Conferma la modifica corrente prima di spostarsi.                                                               |
| `R`                    | Passa alla vista **Resources**.                                                                                                                     |
| `C`                    | Passa alla vista **Calendars**.                                                                                                                     |
| `T`                    | Passa alla vista **Tasks**.                                                                                                                         |
| `P`                    | Mostra la finestra **Project Properties**.                                                                                                           |
| `I`                    | Ingrandisci il diagramma di Gantt.                                                                                                                       |
| `O`                    | Riduci il diagramma di Gantt.                                                                                                                      |
| `L`                    | Collega le attività selezionate.                                                                                                                              |
| `U`                    | Scollega le attività selezionate (rimuovi la dipendenza tra esse).                                                                                       |
| `Numpad +`             | Ingrandisci il diagramma di Gantt.                                                                                                                       |
| `Numpad -`             | Riduci il diagramma di Gantt.                                                                                                                      |
| `Esc`                  | Esci dalla modalità a schermo intero; annulla le finestre di dialogo.                                                                                                            |
| `Ctrl + Enter`         | Conferma la finestra di dialogo corrente.                                                                                                                       |
| `Ctrl + Alt + Shift + H` | Mostra la cronologia delle versioni.                                                                                                           |
