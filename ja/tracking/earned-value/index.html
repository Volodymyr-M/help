アーンドバリュー #

アーンドバリューマネジメント（EVM）は、スコープ、スケジュール、コストの測定を組み合わせて、プロジェクトが予定通り・予算通りに進んでいるかを評価します。ステータス日付とベースラインを設定すれば、Inganttが業界標準のEVM指標を計算します。

ステータス日付 #

Status Dateは、進捗追跡とアーンドバリュー計算に使用される日付を定義します。設定すると、ガントチャート上にその日付の位置に緑色の垂直線が表示されます。

ステータス日付が設定されていない場合、計算には現在の日付が使用されます。ステータス日付はProjectメニューからアクセスできるEarned Value Optionsダイアログで設定できます。

アーンドバリューマネジメント #

Inganttは、スコープ、スケジュール、コストのデータを統合してプロジェクトのパフォーマンスを評価するプロジェクト管理技法であるアーンドバリューマネジメントをサポートしています。

アーンドバリューのセットアップ #

  1. 計画されたスケジュールとコストを記録するためにベースラインを作成します。
  2. Earned Value Optionsダイアログ（Projectメニューからアクセス）でStatus Dateを設定するか、現在の日付をデフォルトとして使用します。
  3. 作業の進行に応じてタスクの% Complete値を更新します。

利用可能なEVM指標 #

以下のEVM列をOptionsダイアログからタスクリストに追加できます：

指標 説明
BCWS 予定作業の予算コスト（計画価値）
BCWP 実施作業の予算コスト（アーンドバリュー）
ACWP 実施作業の実績コスト
BAC 完了時予算
EAC 完了時見積もり
VAC 完了時差異
SV スケジュール差異
CV コスト差異
SPI スケジュール効率指標
CPI コスト効率指標
TCPI 残作業効率指標
SV% スケジュール差異率
CV% コスト差異率

アーンドバリュー方式 #

各タスクはBCWP（アーンドバリュー）の計算に2つの方式のいずれかを使用できます：

  • % Complete（デフォルト） — タスクの達成率を使用してアーンドバリューを計算します。
  • Physical % Complete — 個別に追跡される物理的な達成率を使用します。成果物ベースの進捗測定に便利です。

個々のタスクの方式はTask PropertiesダイアログのCost and EVタブで設定します。新規タスクのデフォルトの方式はEarned Value Optionsダイアログで変更できます。

アーンドバリューベースライン #

EVM計算は特定のベースラインを使用します。Earned Value Optionsダイアログで、アーンドバリューに使用するベースライン（Baseline 0からBaseline 10）を選択できます。デフォルトはBaseline 0です。
