コストの理解 #

タスクとリソースにコストを割り当てると、プロジェクトの総額がわかります。ルートサマリータスクに総合計が表示されるので、予算リソースと比較して、プロジェクトが承認された予算内に収まるかどうかを確認できます。

タスクコストの計算方法 #

各タスクのコストは、タスク自体の固定費と割り当てられたリソースのコストの合計です。Inganttは各タスクの総コストを計算し、タスクリストのCost列に表示します。

タスクのCost = タスクのFixed Cost + タスクに割り当てられたリソースのコスト

タスクリストにCost列が表示されない場合は、OptionsダイアログのTask ColumnsタブでCost Columnフラグにチェックが入っていることを確認してください。

サマリータスクのコスト #

各サマリータスクは、Costフィールドにすべてのサブタスクの総コストを表示します。

ただし、通常のタスクと同様に、サマリータスクにも割り当てリソースとFixed Costを設定できます。これらはサブタスクの総コストに加算され、サマリータスクのCostを増加させます。

サマリータスクのCost = すべてのサブタスクの総コスト + サマリータスクのFixed Cost + サマリータスクに割り当てられたリソースのコスト

ルートサマリータスクを使用して、プロジェクト全体の総コストを確認・管理できます。

予算リソース #

Resource Propertiesダイアログで、リソースをBudgetリソースとしてマークできます。予算リソースはプロジェクトレベルで配分された予算総額を表し、ルートサマリータスクにのみ割り当てることができます。

予算リソースは以下から除外されます：

  • スケジューリング
  • コスト合計
  • 過負荷検出
  • リソースの平準化

予算リソースは、ボトムアップのコスト計算とは別の、トップダウンの予算管理を提供します。

通貨 #

プロジェクトがInganttのデフォルトとは異なる通貨を使用する場合、Project PropertiesダイアログのRegionalタブで変更できます。通貨は記号（例：€）、略称（EUR）、または正式名称（ユーロ）で指定します。

同じタブで、Currency Position（値の前または後、スペースの有無）も指定できます。

通貨を変更しても、コスト値は再計算されません。
