# キーボードショートカット

編集、ナビゲーション、ビューの切り替えなど、よく使う操作のキーボードショートカットで作業を効率化しましょう。

> キーボードショートカットは**Android**および**iOS**では利用できません。
>
> **macOS**では、`Ctrl`の代わりに`Cmd`を使用してください。

| ショートカット               | 説明                                                                                                                                       |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| `Ctrl + X`             | 選択した項目をクリップボードに切り取り。                                                                                                              |
| `Ctrl + C`             | 選択した項目をクリップボードにコピー。                                                                                                             |
| `Ctrl + V`             | クリップボードから選択した項目の下に貼り付け。何も選択されていない場合はリストの末尾に貼り付け。                                         |
| `Ctrl + Z`             | 最後の操作を元に戻す。                                                                                                                             |
| `Ctrl + Y`             | 元に戻した操作をやり直す。                                                                                                                      |
| `Ctrl + S`             | ファイルへの変更を保存。                                                                                                                           |
| `Ctrl + O`             | ファイルを開く（デスクトップではネイティブファイルピッカー、**Web**では**Open file**ダイアログ）。                                                                     |
| `Ctrl + P`             | 現在のプロジェクトを印刷。                                                                                                                        |
| `Ctrl + Shift + F`     | **ヘッダー/メニューバー**の表示/非表示を切り替え。                                                                                                            |
| `A`                    | 新しい項目を追加。                                                                                                                   |
| `Del`                  | 選択した項目を削除。                                                                                                                            |
| `Enter`                | 選択した項目の**Properties**ダイアログを表示。複数項目が選択されている場合、最上位の項目のダイアログが表示されます。                       |
| `Arrow Up`             | 前の項目を選択。                                                                                                                         |
| `Arrow Down`           | 次の項目を選択。                                                                                                                             |
| `Shift + Arrow Up/Down` | リストビューで選択範囲を上下に拡張。                                                                                                |
| `Ctrl + Arrow Up`      | 選択したタスクを上に移動。                                                                                                                           |
| `Ctrl + Arrow Down`    | 選択したタスクを下に移動。                                                                                                                         |
| `Ctrl + Arrow Left`    | 選択したタスクのインデントを下げる。                                                                                                                |
| `Ctrl + Arrow Right`   | 選択したタスクのインデントを上げる。                                                                                                                |
| `Tab / Shift + Tab`    | 次または前の編集可能なセルに移動。移動前に現在の編集を確定。                                                               |
| `R`                    | **Resources**ビューに切り替え。                                                                                                                     |
| `C`                    | **Calendars**ビューに切り替え。                                                                                                                     |
| `T`                    | **Tasks**ビューに切り替え。                                                                                                                         |
| `P`                    | **Project Properties**ダイアログを表示。                                                                                                           |
| `I`                    | ガントチャートをズームイン。                                                                                                                       |
| `O`                    | ガントチャートをズームアウト。                                                                                                                      |
| `L`                    | 選択したタスクをリンク。                                                                                                                              |
| `U`                    | 選択したタスクのリンクを解除（依存関係を削除）。                                                                                       |
| `Numpad +`             | ガントチャートをズームイン。                                                                                                                       |
| `Numpad -`             | ガントチャートをズームアウト。                                                                                                                      |
| `Esc`                  | 全画面モードを終了、ダイアログをキャンセル。                                                                                                            |
| `Ctrl + Enter`         | 現在のダイアログを確定。                                                                                                                       |
| `Ctrl + Alt + Shift + H` | バージョン履歴を表示。                                                                                                           |
