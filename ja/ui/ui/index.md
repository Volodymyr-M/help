# 追加と選択

リストやガントチャートで複数の項目を選択し、新しい項目の挿入位置を制御します。

## 複数項目の選択

**Android**および**iOS**では、長押しで追加の項目を選択します。通常のタップより長く指を押さえてください。

**Windows**、**macOS**、**Web**では、以下の方法を使用します：

- `Ctrl`を押しながらクリックして、個別の複数項目を選択。
- `Shift`を押しながらクリックして、現在の選択からクリックした項目までの範囲を選択。

複数選択は、すべてのリスト（タスク、リソース、カレンダー）とガントチャートのタスクバーで機能します。

## 項目の追加と挿入

**+**ボタンは、現在の選択状態に応じて項目を追加または挿入します。

- 項目（タスク、リソース、カレンダー）が選択されている場合、新しい項目はその下に挿入されます。
- 複数の項目が選択されている場合、新しい項目は最後に選択された項目の下に挿入されます。
- 項目が選択されていない場合、新しい項目はリストの末尾に追加されます。
