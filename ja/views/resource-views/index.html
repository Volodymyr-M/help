リソースビュー #

チーム全体の作業負荷配分を確認できます。配分ビューでは誰がいつ何をしているかを表示し、チームプランナーではドラッグして再割り当てでき、リソースグラフではキャパシティを一目で把握できます。

リソース配分 #

Resource Usageビューでは、すべてのリソースの割り当てと、各稼働リソースがタイムライン上の各期間に実施する作業量を確認できます。

ガントチャートと同様に、タイムラインをズームイン・ズームアウトして、より詳細またはより広範なビューを表示できます。

リソースが指定された期間にカレンダーが許容する以上の作業を行わなければならない場合、対応するWorkが赤色でハイライト表示されます。

Resource Usageビューでは、割り当てレベルの作業セルをダブルクリックして作業配分を直接編集できます。作業セルを編集すると、割り当ての作業量配分は自動的に「Contoured」に変更され、カスタムの日ごとの作業データが保存されます。

タスク配分 #

Task UsageビューはResource Usageの逆で、作業配分のタスク中心のビューを表示します。各タスクがトップレベルの行として表示され、そのリソース割り当てがインデントされてその下に表示されます。

左パネルにはタスクと割り当ての詳細が、右パネルには期間にわたる作業量の値が表示されます。タスクレベルの作業量の値は太字で表示されます。過負荷の割り当て作業量の値は赤色でハイライト表示されます。

チームプランナー #

Team Plannerビューは、各行が稼働リソースを表すリソース中心のスケジューリングビューです。タスクバーは共有のタイムスケール上で各リソースの行に配置されます。

主な機能：

  • 水平にドラッグしてタスクを再スケジュール — タスクに新しい位置でStart No Earlier Than制約が設定されます
  • 垂直にドラッグしてタスクを別のリソースに再割り当て
  • 斜めにドラッグして再スケジュールと再割り当てを1回の操作で実行
  • Unassigned Tasksセクションとの間でドラッグして、リソースの割り当てを追加または削除
  • 過負荷のリソースは左パネルに赤色のテキストで表示
  • 100%完了したタスクはドラッグできません

リソースグラフ #

Resource Graphビューは、リソースごとの作業負荷配分のヒストグラムを時間経過とともに表示します。一度に1つの稼働リソースが表示され、各期間に対して垂直バーが描画されます。左パネルの矢印ボタンでリソース間を移動します。

7種類のグラフタイプが利用可能です：

グラフタイプ ��明
Peak Units 期間あたりのリソースキャパシティ使用率（デフォルト）

Percent Allocation Peak Unitsと同じくパーセンテージ表示

Work Availability カレンダー上の総キャパシティ（時間）
