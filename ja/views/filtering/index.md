# タスクのフィルタリング

今すぐ確認する必要のないタスクを非表示にして、重要なことに集中しましょう。完了した作業、非クリティカルタスク、特定の担当者に割り当てられていないタスクを非表示にできます。

**View**メニューの**Filter**オプションで、タスクビューとガントチャートに表示するタスクをフィルタリングできます。

フィルタオプション：

- **Hide complete tasks** — 100%完了したタスクを非表示
- **Hide non-critical tasks** — クリティカルパス上のタスク（およびその祖先サマリータスク）のみを表示
- **Filter by resource** — 特定のリソースに割り当てられたタスクのみを表示

フィルタを適用するには：

1. **View**メニューから**Filter**を選択
2. 目的のフィルタオプションにチェック
3. フィルタリングするリソースを1つ以上選択（任意）
4. **OK**をクリックしてフィルタを適用

フィルタがアクティブな場合、一致するタスクのみがタスクリストとガントチャートに表示されます。いずれかのフィルタがアクティブな場合、ツールバーのフィルタボタンがハイライト表示されます。すべてのフィルタをリセットするには、Filterダイアログの**Clear All**を使用します。

フィルタは、Tasks、Tracking Gantt、Task Board、Network Diagram、Calendar Viewなど、すべてのタスクベースのビューに適用されます。

> フィルタリングは表示のみに影響し、タスクを削除したり変更したりすることはありません。
