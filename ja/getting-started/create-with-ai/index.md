# AIで作成

AIに完全なプロジェクト計画を作成させましょう。プロジェクトの説明を入力するだけで、Inganttがタスク、依存関係、タイムラインを数秒で生成します。

## プロジェクトの作成

AIでプロジェクトスケジュールを作成するには：

1. プロジェクト画面で**+**ボタンをクリックします。すでにプロジェクトを編集している場合は、代わりに**File**メニューを開きます。
2. **Create new with AI**を選択します。
3. ダイアログボックスにプロジェクトの説明を入力します。
4. **CREATE**をクリックします。

![Create new with AI](/images/getting-started/create_with_ai.png)

## 良いプロンプトの書き方

説明は「_iOSフードデリバリーアプリの開発_」のように一般的なものでも、好きなだけ詳細にすることもできます。

たとえば、[はじめに](/ja/getting-started/getting-started/index.md)で手順を追って紹介しているリフォームプロジェクトは、次のようなプロンプトからでも同じスケジュールを生成できます：

> _「私のリフォームプロジェクト」は2027年1月1日に開始します。塗装工が2日間壁を塗装し、その後に大工が2日間キッチンキャビネットを設置します。塗装工は2027年1月4日に休暇を取ります。2027年1月6日は全員の祝日として扱います。_

プロジェクトが大規模な場合は、サブプロジェクトに分割して、それぞれ個別にプロジェクト計画を生成してみてください。

結果は実行のたびに異なる場合がありますので、出力に満足できなければ再度お試しください。
