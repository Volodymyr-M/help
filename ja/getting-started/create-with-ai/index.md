# AIで作成

AIに完全なプロジェクト計画を作成させましょう。プロジェクトの説明を入力するだけで、Inganttがタスク、依存関係、タイムラインを数秒で生成します。
AIでプロジェクトスケジュールを作成するには：

1. プロジェクト画面で**+**ボタンをクリックします。
2. **Create new with AI**を選択します。
3. ダイアログボックスにプロジェクトの説明を入力します。
4. **CREATE**をクリックします。

> 説明は「_iOSフードデリバリーアプリの開発_」のように一般的なものでも、好きなだけ詳細にすることもできます。プロジェクトが大規模な場合は、サブプロジェクトに分割して、それぞれ個別にプロジェクト計画を生成してみてください。結果は実行のたびに異なる場合がありますので、出力に満足できなければ再度お試しください。
