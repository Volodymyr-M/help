# Inganttでできること

[Ingantt](https://ingantt.com){:target="_blank"}は、プロジェクトのスケジューリングと進捗管理アプリです。プロジェクトのデータ（タスク、期間、依存関係、リソース）を入力すると、Inganttが必要な情報を算出します。

#### プロジェクトの計画時

- **いつ完了するか？** — タスクの依存関係から自動計算され、推測ではありません。
- **どのリソースが必要か？** — 各タスクに必要な人員、機材、資材を把握できます。
- **誰が過負荷か？** — どのリソースがいつ、処理しきれない量の作業を抱えているかを確認できます。
- **いくらかかるか？** — リソース単価と固定費に基づくプロジェクト総コストを算出します。
- **どのタスクが最も重要か？** — クリティカルパスにより、遅延がプロジェクト全体に影響するタスクを特定できます。

#### 作業開始後

- **スケジュール通りか？** — 現在のスケジュールの進捗を当初の計画と比較できます。
- **予算内か？** — アーンドバリュー指標を使って、実績コストと計画コストを比較できます。
- **実際の進捗はどの程度か？** — タスクの期間で重み付けされたプロジェクト全体の達成率を確認でき、感覚に頼りません。
- **いくら使って、いくら残っているか？** — コスト、作業量、期間の実績値と残りを把握できます。

タスクの期間を変更したり、依存関係を追加したりすると、スケジュール全体が即座に再計算されます。すべての日付、すべてのリソース負荷、すべてのコストに反映されます。
