# Campos Personalizados

Adicione suas próprias colunas de dados às tarefas — acompanhe departamento, códigos de prioridade, status de aprovação ou qualquer informação específica do projeto junto com os campos nativos.

O Ingantt suporta campos personalizados em tarefas, incluindo os tipos Text, Number, Flag, Date, Start, Finish, Duration e Cost.

Para gerenciar campos personalizados, abra o item **Custom Fields** no menu **Project**. Você pode adicionar definições de campos personalizados, definir aliases (nomes de exibição) e configurar tabelas de pesquisa com valores predefinidos.

Para editar valores de campos personalizados, use a aba **Custom Fields** da caixa de diálogo **Task Properties**. Para adicionar colunas de campos personalizados à tabela de tarefas, use a caixa de diálogo **Options**.

Os campos personalizados também suportam fórmulas (avaliadas automaticamente) e acumulação em [tarefas resumo](/pt/building-schedule/tasks/index.md#tarefas-resumo) — Soma, Máximo, Mínimo, Contagem e Média.

## Códigos de Estrutura

Os códigos de estrutura são campos de classificação hierárquica definidos pelo usuário para tarefas (ex.: códigos de departamento, códigos de localização). Gerencie-os a partir do item **Outline Codes** no menu **Project**.

Cada código de estrutura tem uma máscara de código que define o formato por nível hierárquico e uma tabela de pesquisa com valores permitidos. Atribua valores de códigos de estrutura na aba **Outline Codes** da caixa de diálogo **Task Properties**. Você pode adicionar colunas correspondentes à tabela de tarefas.
