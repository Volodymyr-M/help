# O Que Você Pode Fazer com o Ingantt

O [Ingantt](https://ingantt.com){:target="_blank"} é um aplicativo de agendamento e acompanhamento de projetos. Você insere os dados do seu projeto — tarefas, durações, dependências, recursos — e o Ingantt fornece as respostas.

#### Ao planejar o projeto

- **Quando ele vai terminar?** — calculado automaticamente a partir das dependências entre tarefas, sem estimativas vagas.
- **Quais recursos são necessários?** — quais pessoas, equipamentos e materiais cada tarefa exige.
- **Quem está sobrecarregado?** — quais recursos têm mais trabalho do que conseguem realizar e quando.
- **Quanto vai custar?** — custo total do projeto baseado nas taxas dos recursos e custos fixos.
- **Quais tarefas são mais importantes?** — o caminho crítico mostra onde um atraso vai impactar o projeto inteiro.

#### Quando o trabalho já está em andamento

- **Estamos no prazo?** — compare o progresso atual do cronograma com o plano original.
- **Estamos dentro do orçamento?** — compare os custos reais com os custos planejados usando métricas de Valor Agregado.
- **Qual é o progresso real?** — percentual concluído geral do projeto, ponderado pela duração das tarefas, não por percepção subjetiva.
- **O que já foi gasto e o que resta?** — custo, trabalho e duração reais versus restantes.

Altere a duração de uma tarefa ou adicione uma dependência, e o cronograma inteiro é recalculado instantaneamente — cada data, cada carga de trabalho de recurso, cada custo.
