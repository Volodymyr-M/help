# Criar com IA

Deixe a IA criar um plano de projeto completo para você — basta descrever seu projeto e o Ingantt gera tarefas, dependências e uma linha do tempo em segundos.

## Criando um Projeto

Para criar um cronograma de projeto com IA:

1. Clique no botão **+** na tela de Projetos. Se você já estiver editando um projeto, abra o menu **File** em vez disso.
2. Escolha **Create new with AI**.
3. Insira uma descrição do seu projeto na caixa de diálogo.
4. Clique em **CREATE**.

![Create new with AI](/images/getting-started/create_with_ai.png)

## Escrevendo um Bom Prompt

Sua descrição pode ser tão genérica quanto _"Desenvolvimento de um aplicativo iOS de delivery de comida"_ ou tão detalhada quanto quiser.

Por exemplo, o projeto de reforma que [Primeiros Passos](/pt/getting-started/getting-started/index.md) apresenta passo a passo produz o mesmo cronograma a partir de um prompt como:

> _"Meu projeto de reforma" começa em 1º de janeiro de 2027. O Pintor pinta as paredes por 2 dias, depois o Carpinteiro instala os armários da cozinha por 2 dias após isso. O Pintor tem um dia de férias em 4 de janeiro de 2027. Considere 6 de janeiro de 2027 como um feriado para todos._

Se o seu projeto for grande, tente dividi-lo em subprojetos e gerar planos separados para cada um.

Os resultados podem variar entre execuções, então se não ficar satisfeito com o resultado, tente novamente.
