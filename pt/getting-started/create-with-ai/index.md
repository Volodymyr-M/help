# Criar com IA

Deixe a IA criar um plano de projeto completo para você — basta descrever seu projeto e o Ingantt gera tarefas, dependências e uma linha do tempo em segundos.
Para criar um cronograma de projeto com IA:

1. Clique no botão **+** na tela de Projetos.
2. Escolha **Create new with AI**.
3. Insira uma descrição do seu projeto na caixa de diálogo.
4. Clique em **CREATE**.

> Sua descrição pode ser tão genérica quanto _"Desenvolvimento de um aplicativo iOS de delivery de comida"_ ou tão detalhada quanto quiser. Se o seu projeto for grande, tente dividi-lo em subprojetos e gerar planos separados para cada um. Os resultados podem variar entre execuções, então se não ficar satisfeito com o resultado, tente novamente.
