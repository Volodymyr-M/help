# 추가 및 선택

목록이나 Gantt 차트에서 여러 항목을 선택하고, 새 항목이 삽입되는 위치를 제어합니다.

## 여러 항목 선택

**Android** 및 **iOS**에서는 길게 누르기를 사용하여 추가 항목을 선택합니다. 일반 탭보다 항목에 손가락을 더 오래 올려놓으십시오.

**Windows**, **macOS** 및 **Web**에서는 다음 방법을 사용합니다:

- `Ctrl`을 누른 채 클릭하여 여러 개별 항목을 선택합니다.
- `Shift`를 누른 채 클릭하여 현재 선택 항목부터 클릭한 항목까지의 범위를 선택합니다.

다중 선택은 모든 목록(작업, 자원, 달력)과 Gantt 차트의 작업 막대에서 작동합니다.

## 항목 추가 및 삽입

**+** 버튼은 현재 선택 상태에 따라 항목을 추가하거나 삽입합니다.

- 항목(작업, 자원 또는 달력)이 선택되어 있으면 새 항목이 해당 항목 아래에 삽입됩니다.
- 여러 항목이 선택되어 있으면 새 항목이 마지막으로 선택된 항목 아래에 삽입됩니다.
- 항목이 선택되지 않은 경우 새 항목이 목록 끝에 추가됩니다.
