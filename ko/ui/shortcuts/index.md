# 키보드 단축키

편집, 탐색, 보기 전환 등 자주 사용하는 작업을 키보드 단축키로 빠르게 수행합니다.

> 키보드 단축키는 **Android** 및 **iOS**에서 사용할 수 없습니다.
>
> **macOS**에서는 `Ctrl` 대신 `Cmd`를 사용하십시오.

| 단축키               | 설명                                                                                                                                       |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| `Ctrl + X`             | 선택한 항목을 클립보드로 잘라냅니다.                                                                                                              |
| `Ctrl + C`             | 선택한 항목을 클립보드로 복사합니다.                                                                                                             |
| `Ctrl + V`             | 클립보드의 항목을 선택한 항목 아래에 붙여넣거나, 선택한 항목이 없으면 목록 끝에 붙여넣습니다.                                         |
| `Ctrl + Z`             | 마지막 작업을 실행 취소합니다.                                                                                                                             |
| `Ctrl + Y`             | 마지막으로 취소한 작업을 다시 실행합니다.                                                                                                                      |
| `Ctrl + S`             | 파일에 변경 사항을 저장합니다.                                                                                                                           |
| `Ctrl + O`             | 파일을 엽니다 (데스크톱에서는 기본 파일 선택기, **Web**에서는 **Open file** 대화 상자).                                                                     |
| `Ctrl + P`             | 현재 프로젝트를 인쇄합니다.                                                                                                                        |
| `Ctrl + Shift + F`     | **헤더/메뉴 바** 표시를 전환합니다.                                                                                                            |
| `A`                    | 새 항목을 추가합니다.                                                                                                                   |
| `Del`                  | 선택한 항목을 삭제합니다.                                                                                                                            |
| `Enter`                | 선택한 항목의 **Properties** 대화 상자를 표시합니다. 여러 항목이 선택된 경우 첫 번째 항목의 대화 상자가 표시됩니다.                       |
| `Arrow Up`             | 이전 항목을 선택합니다.                                                                                                                         |
| `Arrow Down`           | 다음 항목을 선택합니다.                                                                                                                             |
| `Shift + Arrow Up/Down` | 목록 보기에서 선택 범위를 위 또는 아래로 확장합니다.                                                                                                |
| `Ctrl + Arrow Up`      | 선택한 작업을 위로 이동합니다.                                                                                                                           |
| `Ctrl + Arrow Down`    | 선택한 작업을 아래로 이동합니다.                                                                                                                         |
| `Ctrl + Arrow Left`    | 선택한 작업의 들여쓰기를 줄입니다.                                                                                                                |
| `Ctrl + Arrow Right`   | 선택한 작업의 들여쓰기를 늘립니다.                                                                                                                |
| `Tab / Shift + Tab`    | 다음 또는 이전 편집 가능한 셀로 이동합니다. 이동 전에 현재 편집을 확정합니다.                                                               |
| `R`                    | **Resources** 보기로 전환합니다.                                                                                                                     |
| `C`                    | **Calendars** 보기로 전환합니다.                                                                                                                     |
| `T`                    | **Tasks** 보기로 전환합니다.                                                                                                                         |
| `P`                    | **Project Properties** 대화 상자를 표시합니다.                                                                                                           |
| `I`                    | Gantt 차트를 확대합니다.                                                                                                                       |
| `O`                    | Gantt 차트를 축소합니다.                                                                                                                      |
| `L`                    | 선택한 작업을 연결합니다.                                                                                                                              |
| `U`                    | 선택한 작업의 연결을 해제합니다 (종속성 제거).                                                                                       |
| `Numpad +`             | Gantt 차트를 확대합니다.                                                                                                                       |
| `Numpad -`             | Gantt 차트를 축소합니다.                                                                                                                      |
| `Esc`                  | 전체 화면 모드 종료, 대화 상자 취소.                                                                                                            |
| `Ctrl + Enter`         | 현재 대화 상자를 확인합니다.                                                                                                                       |
| `Ctrl + Alt + Shift + H` | 버전 기록을 표시합니다.                                                                                                                           |
