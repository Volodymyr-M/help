# 획득가치

획득가치 관리(EVM)는 범위, 일정 및 비용 측정을 결합하여 프로젝트가 일정대로이고 예산 내에 있는지 평가합니다. 상태 날짜와 기준선을 설정하면 Ingantt가 산업 표준 EVM 지표를 계산합니다.

## 상태 날짜

**Status Date**는 진행 추적 및 획득가치 계산에 사용되는 날짜를 정의합니다. 설정되면 해당 날짜에 Gantt 차트에 녹색 수직선이 표시됩니다.

상태 날짜가 설정되지 않은 경우 현재 날짜가 계산에 사용됩니다. **Project** 메뉴에서 접근할 수 있는 **Earned Value Options** 대화 상자에서 상태 날짜를 설정할 수 있습니다.

## 획득가치 관리

Ingantt는 획득가치 관리를 지원합니다 — 범위, 일정 및 비용 데이터를 통합하여 프로젝트 성과를 평가하는 프로젝트 관리 기법입니다.

### 획득가치 설정

1. 계획된 일정과 비용을 캡처하기 위해 [기준선](/ko/tracking/baselines/index.md#기준선-설정)을 생성합니다.
2. **Earned Value Options** 대화 상자(**Project** 메뉴에서 접근)에서 **Status Date**를 설정하거나 현재 날짜로 기본 설정합니다.
3. 작업이 진행됨에 따라 작업의 [완료율](/ko/tracking/progress/index.md#완료율) 값을 업데이트합니다.

### 사용 가능한 EVM 지표

**Options** 대화 상자를 통해 다음 EVM 열을 작업 목록에 추가할 수 있습니다:

| 지표 | 설명 |
|--------|-------------|
| **BCWS** | 예정 작업의 예산 비용 (계획 가치) |
| **BCWP** | 수행 작업의 예산 비용 (획득가치) |
| **ACWP** | 수행 작업의 실제 비용 |
| **BAC** | 완료 시 예산 |
| **EAC** | 완료 시 추정 비용 |
| **VAC** | 완료 시 차이 |
| **SV** | 일정 차이 |
| **CV** | 비용 차이 |
| **SPI** | 일정 성과 지수 |
| **CPI** | 비용 성과 지수 |
| **TCPI** | 완료 성과 지수 |
| **SV%** | 일정 차이 백분율 |
| **CV%** | 비용 차이 백분율 |

### 획득가치 방법

각 작업은 BCWP(획득가치)를 계산하기 위해 두 가지 방법 중 하나를 사용할 수 있습니다:

- **% Complete** (기본값) — 작업의 완료율을 사용하여 획득가치를 계산합니다.
- **Physical % Complete** — 별도로 추적된 물리적 완료율 값을 사용합니다. 산출물 기반 진행 측정에 유용합니다.

**Task Properties** 대화 상자의 **Cost and EV** 탭에서 개별 작업의 방법을 설정합니다. **Earned Value Options** 대화 상자에서 새 작업의 기본 방법을 변경합니다.

### 획득가치 기준선

EVM 계산은 특정 기준선을 사용합니다. **Earned Value Options** 대화 상자에서 획득가치에 사용할 기준선(`Baseline 0`~`Baseline 10`)을 선택할 수 있습니다. 기본값은 `Baseline 0`입니다.
