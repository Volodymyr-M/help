# 배정

자원이 작업에 어떻게 할당되는지 제어합니다 — 누가 무엇을, 얼마만큼의 시간으로, 노력이 어떻게 분배되는지를 관리합니다. 단위, 작업량 윤곽, 초과 근무를 조정하여 팀의 실제 근무 방식에 맞추십시오.

## 자원 배정 및 단위

자원은 **Task Properties** 대화 상자의 **Resources** 탭에서 작업에 배정할 수 있습니다.

자원을 배정하려면 해당 자원 행의 체크박스를 체크합니다. 자원 배정을 해제하려면 체크박스를 해제합니다.

작업 자원 또는 자재 자원의 배정에는 해당 열에 표시되는 **Units**가 있습니다. **Edit** 버튼을 클릭하여 배정의 기본 **Units** 값을 변경합니다.

기본적으로 작업 자원은 자원의 [Max Units](/ko/building-schedule/resources/index.md#최대-단위)(전일 근무 자원의 경우 100%)와 일치하는 단위로 배정됩니다. 이는 자원이 사용 가능한 모든 달력 시간을 해당 작업에 투입함을 의미합니다. 이 값을 원하는 숫자로 변경할 수 있습니다.

기본적으로 자재 자원은 1단위로 배정됩니다. 이는 작업을 완료할 때 해당 자재 1단위가 사용됨을 의미합니다. 단위는 자재에 대해 정의한 것(상자, 갤런, 톤 등)을 나타냅니다. 기본값을 변경하고 원하는 수량의 단위를 설정할 수 있습니다.

## 작업량 윤곽

작업 자원이 작업에 배정되면 노력(작업량)이 **작업량 윤곽**에 따라 작업 기간에 걸쳐 분배됩니다. 기본적으로 작업량은 균등하게 분배되지만(Flat 윤곽), Ingantt는 시간에 따라 노력이 분배되는 방식을 변경하는 여러 윤곽 패턴을 지원합니다:

| 윤곽 | 설명 |
|---------|-------------|
| **Flat** | 전체 기간에 걸쳐 균일한 노력 (기본값) |
| **Back Loaded** | 작업 후반으로 갈수록 노력이 증가 |
| **Front Loaded** | 작업 초반에 노력이 가장 크고 점차 감소 |
| **Double Peak** | 작업 중 두 번의 강도 피크 |
| **Early Peak** | 초반에 피크를 찍은 후 점차 감소 |
| **Late Peak** | 후반에 피크를 향해 점진적으로 증가 |
| **Bell** | 종 모양 곡선 — 중간에 피크 |
| **Turtle** | 완만한 종 모양 곡선 — 더 고른 분배 |

작업량 윤곽은 작업량이 시간 기간에 걸쳐 분배되는 방식에 영향을 미치며 프로젝트 파일을 열고 저장할 때 유지됩니다.

## 배정 지연

작업의 각 자원 배정에는 자원이 작업 시작 날짜를 기준으로 언제 작업을 시작하는지 조정하는 **Delay** 속성이 있습니다. 예를 들어, 작업이 월요일에 시작되고 자원에 2일 지연이 있으면 해당 자원은 수요일에 작업을 시작합니다.

지연은 **Edit Resource Assignment** 대화 상자에서 설정되며 작업 자원 배정에만 적용됩니다. 이를 사용하여 작업에서 자원의 시작 시간을 엇갈리게 할 수 있습니다.

## 초과 근무

작업 자원의 경우 배정의 총 작업량 중 일부를 초과 근무로 지정할 수 있습니다. 초과 근무 작업량은 총 작업량의 일부이며 추가가 아닙니다: **Work = Regular Work + Overtime Work**.

작업 비용은 정규 근무와 초과 근무 부분을 별도로 사용하여 계산됩니다:
**Cost = Regular Work x Standard Rate + Overtime Work x Overtime Rate + Cost Per Use**

Fixed Units 및 Fixed Work 작업의 경우 초과 근무를 입력하면 기간이 정규 작업량만을 기반으로 하므로 작업 기간이 단축됩니다.

**Edit Resource Assignment** 대화 상자에서 초과 근무를 설정합니다. 작업 테이블에서 **Overtime Work**, **Overtime Cost**, **Regular Work** 세 가지 선택적 열을 사용할 수 있습니다.
