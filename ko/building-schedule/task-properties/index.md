# 작업 속성

각 작업에는 일정 방식, 비용 계산 방법, Gantt 차트에서의 표시 방식을 제어하는 속성이 있습니다. **Task Properties** 대화 상자에서 이를 설정합니다.

## 기간

프로젝트를 계획할 때 기간은 추정값으로 입력합니다. 즉, 관련된 모든 자원이 작업을 완료하는 데 소요될 것으로 합리적으로 예상되는 시간입니다.

**기간**과 **작업량**을 혼동하지 마십시오. 예를 들어, 세 명이 작업을 수행하지만 한 시간 만에 완료한다면 작업의 **Duration**을 한 시간으로 설정합니다. 이 세 명이 작업에 배정되면 Ingantt가 **Work** 속성을 세 시간으로 자동 계산합니다.

기간은 **Task Properties** 대화 상자의 **Duration** 필드를 사용하여 변경할 수 있습니다.

기간 추정에 아직 확신이 없는 경우 **Task Properties** 대화 상자에서 **Estimate**로 표시할 수 있습니다. 이렇게 하면 기간에 항상 물음표("**?**")가 표시됩니다. 이 플래그를 체크하거나 해제해도 일정에는 영향을 미치지 않습니다.

요약 작업의 하위 작업 중 하나 이상에 **Estimate**가 체크되어 있으면, 요약 작업의 기간도 **Estimate**로 표시되어 "**?**"가 표시됩니다.

기간은 시간, 일, 주 또는 월 단위로 설정할 수 있습니다. 기본적으로 "1일"은 8시간, "1주"는 5일(40시간), "1개월"은 20일을 의미합니다. 이 기본값은 **Project Properties** 대화 상자의 **Duration** 탭에서 변경할 수 있습니다.

자원 배정, 작업량 또는 기간을 변경하면 작업의 [유형](#유형-및-노력-기반)에 따라 이 중 하나가 재계산됩니다.

## 작업량

작업에 작업 자원(작업을 수행하는 사람 등)이 배정되면 작업의 **Work** 속성이 0보다 커집니다. 이는 모든 자원이 해당 작업에 소비할 시간을 나타냅니다. 예를 들어, **Duration**이 5시간인 작업에 2명의 자원이 배정되어 작업하는 경우, 작업의 **Work**는 10시간입니다.

작업량은 **Task Properties** 대화 상자의 **Work** 필드를 사용하여 변경할 수 있습니다.

기간과 마찬가지로 작업량도 **Project Properties** 대화 상자의 **Duration** 탭 정의를 사용하여 시간, 일, 주 또는 월 단위로 지정할 수 있습니다. 작업량의 기본 표시 형식은 **Time** 탭에서 변경할 수 있습니다.

자원 배정, 작업량 또는 기간을 변경하면 작업의 [유형](#유형-및-노력-기반)에 따라 이 중 하나가 재계산됩니다.

## 마감일

특정 날짜까지 작업을 완료해야 하는 경우가 있으며, 이를 일반적으로 마감일이라고 합니다.

작업의 마감일은 **Task Properties** 대화 상자의 **Deadline** 필드를 사용하여 지정할 수 있습니다.

마감일은 참조용이며 일정에 영향을 미치지 않습니다.

마감일은 Gantt 차트에서 특수 아이콘으로 표시됩니다.

> 일정상 작업이 지정된 마감일보다 늦게 완료되는 경우, Ingantt는 작업 목록에 아이콘을 표시하고 탐색 서랍에 해당 작업 수를 카운트합니다.

![마감일](/images/building-schedule/tasks/deadline.png)

> 루트 요약 작업에 마감일을 설정하여 전체 프로젝트에 대한 마감일을 지정할 수 있습니다. 루트 요약 작업이 **Options** 대화 상자에서 표시되도록 설정되어 있는지 확인하십시오.

## 마일스톤

**Task Properties** 대화 상자에서 **Milestone** 체크박스를 체크하여 모든 작업을 마일스톤으로 표시할 수 있습니다. 이는 기간이나 일정에 영향을 주지 않지만 Gantt 차트에서 아이콘으로 표시됩니다.

![마일스톤](/images/building-schedule/tasks/milestone.png)

작업의 **Duration**을 0으로 지정하면 변경 사항을 저장할 때 자동으로 **Milestone**으로 표시됩니다.

## 유형 및 노력 기반

작업 자원 배정(또는 배정된 작업 자원의 단위), 작업량 및 기간은 서로 종속됩니다. 이 중 하나를 변경하면 나머지가 그에 맞게 재계산되어야 합니다. 작업의 **Type**(**Effort Driven** 플래그의 도움으로)은 나머지 두 속성 중 어느 것이 변경되지 않는지를 정의하여 하나만 재계산되도록 합니다.

예를 들어, **Type**을 **Fixed units**(기본값)로 설정하면 Duration을 변경할 때 Work가 자동으로 재계산됩니다.

| 유형               | 설명                                             |
|--------------------|---------------------------------------------------------|
| **Fixed units**    | Duration을 변경하면: Work가 재계산됩니다.         |
|                    | Work를 변경하면: Duration이 재계산됩니다.         |
|                    | Units를 변경하면:                                  |
|                    | - **Effort Driven**이 설정된 경우: Duration이 재계산됩니다. |
|                    | - **Effort Driven**이 설정되지 않은 경우: Work가 재계산됩니다. |
| **Fixed duration** | Duration을 변경하면: Work가 재계산됩니다.         |
|                    | Work를 변경하면: Units가 재계산됩니다.           |
|                    | Units를 변경하면: Work가 재계산됩니다.            |
| **Fixed work**     | Duration을 변경하면: Units가 재계산됩니다.       |
|                    | Work를 변경하면: Duration이 재계산됩니다.         |
|                    | Units를 변경하면: Duration이 재계산됩니다.        |

즉, **Type**은 세 가지 속성 중 하나를 고정하고, **Effort Driven** 플래그는 나머지 두 속성 중 Work를 변경하지 않을지 여부를 정의합니다.

> **Type** 및 **Effort Driven** 설정은 항상 Fixed duration이고 노력 기반이 아닌 [요약 작업](/ko/building-schedule/tasks/index.md#요약-작업)에는 사용할 수 없습니다.

## 메모

**Task Properties** 대화 상자의 **Notes** 탭에 있는 **Notes** 필드에 작업에 대한 텍스트를 추가할 수 있습니다. 작업 설명, 연락처 정보, 아이디어 또는 기타 텍스트 데이터에 사용하십시오.

작업에 **Notes** 필드가 채워져 있으면 작업 목록에 특수 아이콘이 표시됩니다. Windows, macOS 및 웹에서는 아이콘 위에 마우스를 올리면 메모가 표시됩니다. 모바일에서는 **Task Properties** 대화 상자를 열어 전체 메모를 확인하십시오.

## 하이퍼링크

**Task Properties** 대화 상자의 **Notes** 탭에 있는 **Hyperlink** 필드를 사용하여 작업에 URL을 첨부할 수 있습니다. 하이퍼링크가 있는 작업은 작업 목록에 링크 아이콘이 표시됩니다. 링크 아이콘을 클릭하면 브라우저에서 URL이 열립니다.

## 막대 숨기기 및 롤업

**Task Properties** 대화 상자의 **Visual** 탭에서:

- **Hide bar** — 작업 목록에서 행은 유지하면서 Gantt 차트에서 작업 막대를 숨깁니다. 보이지 않는 막대 영역은 여전히 클릭 및 컨텍스트 메뉴에 반응합니다.
- **Rollup** — 하위 작업의 막대를 상위 요약 작업 행의 Gantt 차트에 표시합니다. 요약 작업이 접혀 있을 때 축약된 보기를 제공합니다.

이 옵션은 **Edit > Display** 하위 메뉴 또는 마우스 우클릭 컨텍스트 메뉴에서도 전환할 수 있습니다.
