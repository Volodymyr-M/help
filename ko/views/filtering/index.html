작업 필터링 #

지금 볼 필요가 없는 작업을 숨겨 중요한 사항에 집중합니다 — 완료된 작업, 비주공정 작업 또는 특정 사람에게 배정되지 않은 작업을 숨깁니다.

View 메뉴의 Filter 옵션을 사용하여 작업 보기와 Gantt 차트에 표시되는 작업을 필터링할 수 있습니다.

필터 옵션은 다음과 같습니다:

  • Hide complete tasks — 100% 완료된 작업 숨기기
  • Hide non-critical tasks — 주공정경로의 작업(및 상위 요약 작업)만 표시
  • Filter by resource — 특정 자원에 배정된 작업만 표시

필터를 적용하려면:

  1. View 메뉴에서 Filter를 선택합니다.
  2. 원하는 필터 옵션을 체크합니다.
  3. 하나 이상의 자원을 선택하여 필터링합니다 (선택 사항).
  4. OK를 클릭하여 필터를 적용합니다.

필터가 활성화되면 작업 목록과 Gantt 차트에 일치하는 작업만 표시됩니다. 필터가 활성화되어 있으면 도구 모음의 필터 버튼이 강조 표시됩니다. Filter 대화 상자에서 Clear All을 사용하여 모든 필터를 초기화합니다.

필터는 모든 작업 기반 보기에 적용됩니다: Tasks, Tracking Gantt, Task Board, Network Diagram, Calendar View 등.

필터링은 표시에만 영향을 미치며 작업을 삭제하거나 수정하지 않습니다.
