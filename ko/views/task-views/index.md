# 작업 보기

Gantt 차트 외에도 칸반 보드, 월간 달력, 네트워크 다이어그램 또는 타임라인에서 작업을 확인할 수 있습니다. 각 보기는 일정의 다른 측면을 강조합니다.

## Task Board

**Task Board** 보기는 작업을 워크플로우 단계를 나타내는 열에 카드로 구성하여 표시합니다(칸반 방식).

새 프로젝트는 세 개의 기본 열로 시작합니다: "Not Started" (0%), "In Progress" (50%), "Complete" (100%). 각 열에는 완료율 매핑이 있으며, 카드를 열로 드래그하면 작업의 완료율이 자동으로 업데이트됩니다.

열을 추가, 편집, 이름 변경, 재정렬 및 삭제할 수 있습니다. 각 열 하단에 **Add Task** 버튼이 있어 해당 단계에서 직접 작업을 만들 수 있습니다.

비요약 활성 작업 중 "Show on board"가 활성화된 작업만 보드에 표시됩니다. "Show on board" 설정은 **Task Properties** 대화 상자의 **Visual** 탭에서 사용할 수 있습니다.

> 작업 카드를 열 간에 드래그하여 상태를 변경합니다. 작업의 완료율이 다른 곳(대화 상자, 인라인 편집 또는 Update Project)에서 변경되면 카드가 자동으로 해당 열로 이동합니다.

## Network Diagram

**Network Diagram** 보기는 작업을 왼쪽에서 오른쪽으로 흐르는 순서도 레이아웃의 직사각형 노드로 표시합니다. 선행 작업이 없는 작업은 가장 왼쪽 열에 나타나고, 각 후속 작업은 모든 선행 작업 뒤에 나타납니다.

- 일반 작업은 ID, 기간, 시작/완료 날짜, 자원을 표시합니다.
- 요약 작업은 ID, 기간, 시작/완료 날짜, 완료율을 표시합니다.
- 마일스톤은 ID와 시작 날짜를 표시합니다.
- 주공정 작업은 빨간색 헤더와 테두리를 가집니다.
- 진행 상황은 대각선 오버레이로 표시됩니다: 진행 중인 작업에는 한 줄, 완료된 작업에는 교차 줄

요약 작업을 접기/펼치기, 클릭하여 선택, 더블클릭하여 편집, 마우스 우클릭으로 컨텍스트 메뉴를 사용할 수 있습니다. 네 가지 링크 유형 모두에 대해 노드 간 종속성 화살표가 그려집니다.

## Calendar View

**Calendar View**는 작업이 예정된 날짜에 걸쳐 수평 막대로 표시되는 월간 달력 격자를 표시합니다.

- 헤더의 토글을 사용하여 **Month**와 **Week** 보기 간에 전환
- **Work week** 모드를 활성화하면 평일만 표시(5일 보기)
- 이전/다음 버튼으로 탐색하거나 오늘로 이동
- 비근무일은 구별되는 배경으로 표시
- 마일스톤은 다이아몬드 아이콘으로 표시
- 빈 날짜를 더블클릭하면 해당 날짜에 1일 작업이 새로 생성

주 경계를 넘는 작업은 다음 행으로 줄바꿈됩니다. 같은 날에 여러 작업이 겹치면 수직으로 쌓입니다.

## Timeline

**Timeline**은 Gantt 차트 위에 나타나는 접을 수 있는 패널로, 사용자가 선택한 작업을 수평 시간 축에 간략하게 표시합니다.

타임라인에 작업을 추가하려면 마우스 우클릭 후 **Display on Timeline**을 선택하거나, Task Properties 대화 상자의 **Visual** 탭에서 옵션을 체크합니다.

타임라인의 작업은 막대 또는 콜아웃(수직 연결선이 있는 텍스트 레이블)으로 표시할 수 있습니다. 타임라인 패널의 수평 구분선 위 또는 아래로 작업 막대를 드래그하여 모드를 전환할 수 있습니다. 타임라인에는 오늘 표시와 드래그하여 Gantt 차트를 스크롤할 수 있는 뷰포트 오버레이도 표시됩니다.

**View** 메뉴에서 Timeline을 전환합니다.

## Tracking Gantt

**Tracking Gantt** 보기는 표준 Tasks 보기와 유사하지만 주공정경로 강조가 항상 활성화되어 있고 Baseline 0이 저장된 경우 표시됩니다. Tracking Gantt에서 다른 보기로 이동하면 주공정경로와 기준선 표시 설정이 이전 상태로 복원됩니다.

기준선 계획 대비 프로젝트 진행 상황을 모니터링하는 데 Tracking Gantt를 사용하십시오.
