# 자원 보기

팀 전체의 업무량 분포를 확인합니다. 사용량 보기에서 누가 언제 무엇을 하는지 보고, Team Planner에서 드래그하여 재배정하고, Resource Graph에서 용량을 한눈에 파악합니다.

## Resource Usage

**Resource Usage** 보기에서 모든 자원 배정과 각 작업 자원이 타임라인의 각 기간에 수행하는 [작업량](/ko/building-schedule/task-properties/index.md#작업량)을 확인할 수 있습니다.

Gantt 차트와 마찬가지로 타임라인을 확대하거나 축소하여 더 상세하거나 간략한 보기를 확인할 수 있습니다.

자원이 주어진 기간 동안 달력이 허용하는 것보다 더 많은 작업을 수행해야 하는 경우, 해당 **Work**가 빨간색으로 강조 표시됩니다.

Resource Usage 보기에서 배정 수준의 작업량 셀을 더블클릭하여 작업량 분배를 직접 편집할 수 있습니다. 작업량 셀을 편집하면 배정의 작업량 윤곽이 자동으로 "Contoured"로 변경되고 사용자 정의 일별 작업량 데이터가 저장됩니다.

## Task Usage

**Task Usage** 보기는 Resource Usage의 반대입니다 — 작업 중심의 작업량 분배 보기를 표시합니다. 각 작업이 최상위 행으로 나타나고 그 아래에 자원 배정이 들여쓰기되어 표시됩니다.

왼쪽 패널에는 작업 및 배정 세부 정보가, 오른쪽 패널에는 시간 기간에 걸쳐 분배된 작업량 값이 표시됩니다. 작업 수준의 작업량 값은 굵은 글씨로 표시됩니다. 과배정된 배정 작업량 값은 빨간색으로 강조 표시됩니다.

## Team Planner

**Team Planner** 보기는 각 행이 작업 자원을 나타내는 자원 중심 일정 보기입니다. 작업 막대는 공유 시간 척도를 따라 각 자원의 행에 배치됩니다.

주요 기능:

- **수평 드래그**로 작업 일정 변경 — 작업에 새 위치의 Start No Earlier Than 제약 조건이 적용됩니다.
- **수직 드래그**로 다른 자원에 작업 재배정
- **대각선 드래그**로 한 번에 일정 변경과 재배정
- **Unassigned Tasks** 섹션으로 또는 해당 섹션에서 드래그하여 자원 배정 추가 또는 제거
- 과배정된 자원은 왼쪽 패널에 빨간색 텍스트로 표시
- 100% 완료된 작업은 드래그할 수 없음

## Resource Graph

**Resource Graph** 보기는 시간에 따른 자원별 업무량 할당 히스토그램을 표시합니다. 한 번에 하나의 작업 자원을 표시하며 각 기간에 대한 수직 막대가 있습니다. 왼쪽 패널의 화살표 버튼을 사용하여 자원 간에 이동합니다.

7가지 그래프 유형을 사용할 수 있습니다:

| 그래프 유형 | 설명 |
|------------|-------------|
| **Peak Units** | 기간별 자원 용량 사용 비율 (기본값) |
| **Work** | 기간별 작업 시간 |
| **Overallocation** | 용량을 초과하는 시간만 표시 |
| **Percent Allocation** | Peak Units와 동일하며 백분율 표시 |
| **Remaining Availability** | 미사용 용량(시간) |
| **Work Availability** | 총 달력 용량(시간) |
| **Unit Availability** | 유효 최대 단위 백분율 |

파란색 막대는 용량 내의 작업을 나타내고, 빨간색 막대는 용량을 초과하는 과배정 부분을 나타냅니다.
