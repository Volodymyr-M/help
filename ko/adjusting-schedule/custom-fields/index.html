사용자 정의 필드 #

작업에 자체 데이터 열을 추가하여 부서, 우선순위 코드, 승인 상태 또는 기본 제공 필드와 함께 프로젝트별 정보를 추적합니다.

Ingantt는 Text, Number, Flag, Date, Start, Finish, Duration, Cost 필드 유형을 포함한 작업의 사용자 정의 필드를 지원합니다.

사용자 정의 필드를 관리하려면 Project 메뉴에서 Custom Fields 항목을 여십시오. 사용자 정의 필드 정의를 추가하고, 별칭(표시 이름)을 설정하며, 미리 정의된 값이 포함된 조회 테이블을 구성할 수 있습니다.

사용자 정의 필드 값을 편집하려면 Task Properties 대화 상자의 Custom Fields 탭을 사용하십시오. 작업 테이블에 사용자 정의 필드 열을 추가하려면 Options 대화 상자를 사용하십시오.

사용자 정의 필드는 수식(자동 평가) 및 요약 작업에 대한 롤업(Sum, Max, Min, Count, Average)도 지원합니다.

개요 코드 #

개요 코드는 작업에 대한 사용자 정의 계층적 분류 필드입니다(예: 부서 코드, 지역 코드). Project 메뉴의 Outline Codes 항목에서 관리합니다.

각 개요 코드에는 계층 수준별 형식을 정의하는 코드 마스크와 허용 값의 조회 테이블이 있습니다. Task Properties 대화 상자의 Outline Codes 탭에서 개요 코드 값을 지정합니다. 해당 열을 작업 테이블에 추가할 수 있습니다.
