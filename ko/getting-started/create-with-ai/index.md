# AI로 생성하기

AI가 완전한 프로젝트 계획을 대신 만들어 드립니다 — 프로젝트를 설명하기만 하면 Ingantt가 몇 초 만에 작업, 종속성, 타임라인을 생성합니다.
AI로 프로젝트 일정을 생성하려면:

1. 프로젝트 화면에서 **+** 버튼을 클릭합니다.
2. **Create new with AI**를 선택합니다.
3. 대화 상자에 프로젝트에 대한 설명을 입력합니다.
4. **CREATE**를 클릭합니다.

> 설명은 _"iOS 음식 배달 앱 개발"_처럼 일반적이어도 되고 원하는 만큼 상세하게 작성해도 됩니다. 프로젝트가 큰 경우 하위 프로젝트로 나누어 각각에 대해 별도의 프로젝트 계획을 생성해 보십시오. 실행할 때마다 결과가 다를 수 있으므로 결과가 만족스럽지 않으면 다시 시도해 보십시오.
