AI로 생성하기 #

AI가 완전한 프로젝트 계획을 대신 만들어 드립니다 — 프로젝트를 설명하기만 하면 Ingantt가 몇 초 만에 작업, 종속성, 타임라인을 생성합니다.

프로젝트 만들기 #

AI로 프로젝트 일정을 생성하려면:

  1. 프로젝트 화면에서 + 버튼을 클릭합니다. 이미 프로젝트를 편집 중이라면 대신 File 메뉴를 여십시오.
  2. Create new with AI를 선택합니다.
  3. 대화 상자에 프로젝트에 대한 설명을 입력합니다.
  4. CREATE를 클릭합니다.

좋은 프롬프트 작성하기 #

설명은 _"iOS 음식 배달 앱 개발"_처럼 일반적이어도 되고 원하는 만큼 상세하게 작성해도 됩니다.

예를 들어, 시작하기에서 단계별로 안내하는 리모델링 프로젝트는 다음과 같은 프롬프트로도 동일한 일정을 생성할 수 있습니다:

"나의 리모델링 프로젝트"는 2027년 1월 1일에 시작됩니다. 페인터(Painter)가 2일 동안 벽을 칠하고, 그다음 카펜터(Carpenter)가 2일 동안 주방 수납장을 설치합니다. 페인터는 2027년 1월 4일에 휴가가 있습니다. 2027년 1월 6일은 모든 사람에게 공휴일로 처리하십시오.

프로젝트가 큰 경우 하위 프로젝트로 나누어 각각에 대해 별도의 프로젝트 계획을 생성해 보십시오.

실행할 때마다 결과가 다를 수 있으므로 결과가 만족스럽지 않으면 다시 시도해 보십시오.
