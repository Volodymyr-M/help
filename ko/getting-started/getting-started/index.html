시작하기 #

여기에서 첫 번째 프로젝트 일정을 만들어 보십시오. 몇 분이면 타임라인 위에 작업을 배치하고, 서로 연결하고, 자원을 배정하여 프로젝트가 언제 완료되는지 확인할 수 있습니다.

모바일 기기에서는 클릭 대신 탭, 더블클릭 대신 더블탭, 마우스 우클릭 대신 길게 누르기를 사용하십시오.

기본 일정 만들기 #

소규모 주택 리모델링 프로젝트에 두 개의 작업이 있고, 하나가 다른 하나 다음에 진행된다고 가정해 보겠습니다.

Create new를 클릭하면 왼쪽에 작업 목록, 오른쪽에 Gantt 차트가 있는 분할 화면이 나타납니다. 오른쪽 하단의 큰 + 버튼을 클릭하여 작업을 하나씩 추가하십시오. Add Task 대화 상자에서 각 작업의 Name과 예상 Duration을 지정합니다.

왼쪽 목록에서 두 작업을 모두 선택한 다음 도구 모음의 Link tasks 버튼을 클릭합니다. 이렇게 하면 작업이 연결되고 Gantt 차트가 업데이트됩니다.

  • Android 및 iOS에서 여러 작업을 선택하려면 목록이나 Gantt 차트에서 각 작업을 탭할 때 손가락을 조금 더 길게 누르십시오.
  • 다른 플랫폼에서는 키보드의 Ctrl 또는 Shift를 누른 채로 목록이나 Gantt 차트에서 작업을 클릭하십시오.

작업이 연결되었으므로 첫 번째 작업의 기간을 변경해 보십시오. 목록이나 Gantt 차트에서 해당 작업을 더블클릭하고 Task Properties 대화 상자에서 Duration 필드를 편집합니다. 이 변경이 두 번째 작업을 이동시켜 일정에 어떤 영향을 미치는지 확인할 수 있습니다.

프로젝트 이름 및 시작 날짜 설정 #

프로젝트 시작일이 Ingantt의 기본값과 다를 수 있습니다.

탐색 서랍을 사용하거나, 웹 및 Windows에서는 헤더의 프로젝트 이름을 클릭하여 Project Properties를 여십시오.

Project Start Date 필드에서 새 프로젝트 시작 날짜를 설정하고, 함께 프로젝트의 Name도 변경하십시오.

기본 일정만으로도 Gantt 차트를 보면 프로젝트가 언제 끝나는지 알 수 있습니다.

자원 추가 및 배정 #

프로젝트의 각 작업을 다른 사람이 담당한다고 가정해 보겠습니다.

탐색 서랍을 열고 Resources 보기로 전환하십시오.

큰 + 버튼을 클릭하여 Add Resource 대화 상자를 열고 자원의 Name을 입력한 후 저장합니다. 같은 단계를 반복하여 다른 작업 자원을 추가하십시오.

탐색 서랍을 사용하여 Tasks 보기로 돌아가십시오. 목록이나 Gantt 차트에서 첫 번째 작업을 더블클릭하여 Task Properties 대화 상자를 여십시오.

Resources 탭으로 전환하고, 첫 번째 자원에 체크하여 이 사람을 작업에 배정한 후 저장합니다.

두 번째 작업과 두 번째 자원에 대해서도 같은 단계를 반복하십시오.

휴가 설정 #

프로젝트 진행 중에 사람마다 타임라인에 영향을 미치는 다양한 일정이 있을 수 있습니다.

프로젝트의 첫 번째 작업을 담당하는 자원이 해당 작업 기간 중 하루 휴가가 필요하다고 가정해 보겠습니다.

이를 구성하려면 탐색 서랍을 사용하여 Resources 보기로 이동하십시오.

자원을 더블클릭하여 Resource Properties를 확인합니다. Calendar 탭으로 이동하여 Add exception 버튼을 클릭합니다. 프로젝트 기간 중 하루를 선택하고, 근무 시간 목록을 비워 둔 채 저장합니다.

탐색 서랍을 사용하여 Tasks로 돌아가 일정이 어떻게 변경되었는지 확인하십시오. 두 번째 작업이 첫 번째 작업에 대한 종속성으로 인해 이동되었습니다.

휴가뿐만 아니라 개인 일정의 다른 변경 사항도 구성할 수 있습니다. 예를 들어, 예외에 근무 시간을 추가하면 해당 자원이 특정 날짜에 특정 시간에만 근무하도록 지정할 수 있습니다.

공휴일 설정 #

프로젝트 기간 중 아무도 일하지 않는 공휴일이 있다고 가정해 보겠습니다.

휴가는 각 자원의 달력을 통해 구성됩니다. 반면 모든 사람에게 영향을 미치는 날은 프로젝트 달력에서 구성합니다. 기본 프로젝트 달력은 Standard이며, 이미 주말에는 작업이 수행되지 않도록 설정되어 있습니다.

공휴일을 구성하려면 탐색 서랍을 사용하여 Calendars 보기로 이동하십시오. Standard를 더블클릭하여 Calendar Properties를 확인합니다. Add exception 버튼을 클릭합니다. 프로젝트 기간 중 하루를 선택하고, 근무 시간 목록을 비워 둔 채 저장합니다.

탐색 서랍을 사용하여 Tasks 보기로 돌아가 프로젝트가 어떻게 변경되었는지 확인하십시오. 작업이 이동되었으며, Gantt 차트에서 공휴일은 주말과 유사하게 표시됩니다.

같은 방법을 사용하여 공휴일뿐만 아니라 모든 사람의 일정에 대한 다른 변경 사항도 구성할 수 있습니다. 예를 들어, 예외에 근무 시간을 추가하면 해당 날짜에 모든 사람이 특정 시간에만 근무하도록 지정할 수 있습니다.

Calendar Properties에서 각 요일을 나타내는 탭도 확인할 수 있습니다. 팀 전체가 금요일에 일찍 퇴근하거나 토요일에 근무하는 경우, 해당 탭에서 구성할 수 있습니다.
