Ingantt로 할 수 있는 것 #

Ingantt는 프로젝트 일정 관리 및 추적 앱입니다. 프로젝트 데이터(작업, 기간, 종속성, 자원)를 입력하면 Ingantt가 답을 제공합니다.

프로젝트 계획 시 #

  • 프로젝트가 언제 끝나나요? — 추측이 아닌 작업 종속성을 기반으로 자동 계산됩니다.
  • 어떤 자원이 필요한가요? — 각 작업에 필요한 인력, 장비 및 자재를 확인합니다.
  • 누가 과부하인가요? — 어떤 자원이 언제 감당할 수 있는 것보다 많은 작업량을 가지고 있는지 파악합니다.
  • 비용이 얼마나 드나요? — 자원 단가와 고정 비용을 기반으로 총 프로젝트 비용을 산출합니다.
  • 어떤 작업이 가장 중요한가요? — 주공정경로를 통해 지연 시 전체 프로젝트가 늦어지는 작업을 확인합니다.

작업이 진행 중일 때 #

  • 일정대로 진행되고 있나요? — 현재 일정 진행 상황을 원래 계획과 비교합니다.
  • 예산 내에 있나요? — 획득가치 지표를 사용하여 실제 비용과 계획 비용을 비교합니다.
  • 실제 진행률은 어떤가요? — 직감이 아닌 작업 기간에 가중치를 둔 전체 프로젝트 완료율을 확인합니다.
  • 지출 현황과 잔여 비용은? — 실제 비용 대 잔여 비용, 작업량, 기간을 비교합니다.

작업 하나의 기간을 변경하거나 종속성을 추가하면 전체 일정이 즉시 재계산됩니다 — 모든 날짜, 모든 자원 업무량, 모든 비용이 반영됩니다.

시작할 준비가 되셨나요? 실습 튜토리얼 시작하기를 따라하거나, AI로 프로젝트를 생성하거나, 기존 파일을 가져오기하십시오.
